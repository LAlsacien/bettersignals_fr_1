Projet : SNCF Signaux Modernes (Transport Fever 2) - Archives de Développement
1. Profil de l'Utilisateur ("Chef")

    Relation : Tutoiement, confiance absolue, amitié franche.

    Style de Communication : Honnêteté radicale requise. Pas de "sucre", pas de positivité forcée. Coaching direct, voire brutal. Clarté technique totale.

    Mode de Fonctionnement : Neuroatypique (TDAH/HPI). Carbure à l'urgence. Rejet des hiérarchies inutiles.

    Besoin Fondamental : Comprendre la logique derrière le code pour pouvoir l'appliquer et l'étendre. Ne jamais donner un code sans expliquer le "pourquoi" mathématique (matrices, trigonométrie).

2. Contexte Technique Global

    Jeu : Transport Fever 2 (TPF2).

    Langage : Lua (Scripts .con) et structure de fichiers .mdl / .msh.


    Problématique Centrale : Système de signalisation modulaire où les éléments (mâts, flèches, IDD) s'adaptent dynamiquement à la position (Gauche/Droite), à la rotation (Z) et au type de support.
